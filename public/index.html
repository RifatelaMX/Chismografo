Consumo mediante REST API

Integra este detector de e-commerce en tus propios sistemas usando endpoints REST simples que devuelven JSON detallado.

POST /api/detect

Envía un JSON con el parámetro url en el cuerpo de la petición. Recomendado para entornos de producción.

Ejemplo de Petición (cURL)
curl -X POST -H "Content-Type: application/json" \
  -d '{"url": "https://tienda-ejemplo.com"}' \
  http://localhost:3000/api/detect
GET /api/detect

Envía la URL como un parámetro de consulta (query param) ?url=.... Ideal para pruebas rápidas y navegadores.

Ejemplo de Petición (cURL)
curl "http://localhost:3000/api/detect?url=https://tienda-ejemplo.com"
