🛠️ Arquitectura y Setup

Instrucciones para correr el Chismógrafo en tu máquina.

Requisitos

  • Node.js (v18+)
  • npm o yarn

Instalación

git clone https://github.com/RifatelaMX/Chismografo.git
cd Scrapper
npm install
npm run dev

💻 CLI Chismógrafo

Cómo usar la consola para sacar chismes.

# Vincular comando localmente
npm link

# Escanear dominio
chismografo test-domain shopify.com

# Escanear con atributo específico
chismografo test-domain shopify.com --attr cms

# Asistente interactivo
chismografo add-cms
chismografo add-app
chismografo add-infra

🌐 API REST

Endpoints para conectar con el motor de escaneo.

POST /api/detect

Escanear sitio completo (Cuerpo JSON).

curl -X POST -H "Content-Type: application/json" -d '{"url": "shopify.com"}' http://localhost:3000/api/detect
GET /api/detect

Escanear sitio completo (Query param).

curl "http://localhost:3000/api/detect?url=shopify.com"
POST /api/pagespeed

Lanzar speed test de Google PageSpeed.

curl -X POST -H "Content-Type: application/json" -d '{"url": "shopify.com"}' http://localhost:3000/api/pagespeed
